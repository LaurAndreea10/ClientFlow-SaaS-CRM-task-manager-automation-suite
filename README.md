## ✨ About the Project

**ClientFlow SaaS** is an all-in-one platform built for **freelancers, agencies, consultants, and small teams** who want better control over their daily operations.

It combines the most important workflow tools into a single dashboard:

- 👥 Client management  
- ✅ Task planning  
- 📅 Calendar scheduling  
- 🔔 Reminders & automations  
- 🧾 Service templates  
- ⏱️ Time tracking  
- 💸 Invoice generation  
- 🌐 Client portal  
- 🖼️ Availability image generator  
- ✍️ Content adaptation tools  

The goal is simple: **less chaos, more clarity, faster execution**.

## 🧪 Product Reality Check (MVP Demo)

This repository currently ships a **frontend MVP demo / prototype** built in a single-page browser app.

### ✅ Implemented now
- Core dashboard navigation and module views
- Local client/task/reminder/invoice management
- Local time tracking and service templates
- Bilingual RO/EN UI switch
- Browser persistence via LocalStorage

### 🧩 Simulated / mocked in current demo
- Reminder automations (stored locally as drafts)
- Client portal links (preview simulation)
- Invoice PDF export (print-ready browser flow)
- Content studio generation (local templates, no AI backend)

### 🗺️ Planned next
- Authentication
- Backend APIs
- Cloud database
- Real notifications
- Public secure client portal
- Team roles and permissions

## 🧩 Features

### 👥 Client Management
Manage all client information from one place.

- Add and organize clients
- Store emails, phone numbers, company details
- Save internal notes
- Keep communication context centralized

### ✅ Task Management
Track work efficiently and clearly.

- Create tasks with title, details, date ranges, and deadlines
- Assign status:
  - `Pending`
  - `On Hold`
  - `Finished`
- Use priority scoring for better planning

### 📅 Calendar View
See your workload visually.

- Monthly overview
- Deadline tracking
- Task interval visibility
- Quick planning experience

### 🔔 Reminder Center & Automations
Stay ahead of deadlines and follow-ups.

- Create manual reminders
- Use preset automation flows
- Prepare the app for push/email reminder logic

### 🧾 Service Templates
Standardize your offers and workflow.

- Save reusable service templates
- Define pricing, duration, and deliverables
- Speed up sales and onboarding

### ⏱️ Time Tracking
Measure productivity and bill more accurately.

- Log hours worked
- Track time by task and client
- Build a better workflow overview

### 💸 Invoicing
Handle billing directly inside the platform.

- Generate invoices
- Preview invoice layout
- Export invoice as PDF

### 🖼️ Availability Generator
Create branded schedule visuals in seconds.

- Upload client logo
- Keep brand colors consistent
- Generate availability poster
- Export as PNG or PDF

### 🌐 Client Portal
Give clients a simple overview of their work.

- Generate simulated public links
- Preview portal layout
- Show tasks and invoices in one place

### ✍️ Content Studio
Repurpose one idea across multiple platforms.

- Rewrite content for:
  - Instagram
  - TikTok
  - X / Twitter
  - LinkedIn
  - Facebook
- Choose tone of voice
- Generate:
  - captions
  - CTA hooks
  - hashtags
